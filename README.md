# GitHub Spec Kit Guide

This repository contains research, documentation, and resources for the [GitHub Spec Kit](https://github.com/github/spec-kit).

## Structure

- **raw_data/**: Unprocessed, unsorted, and unapproved research data. Usage is restricted (see `AGENTS.md`).
- **docs/**: Official documentation, principles (like SDD), and theory.
- **workflows/**: Proven workflows for using Spec Kit.
- **commands/**: Prompts and command examples.
- **artifacts/**: Examples of generated documents (plans, specs, tasks).
- **faq/**: Common questions and answers.
- **tricks/**: Tips and tricks for getting the best out of the framework.

## Getting Started with Spec Kit

(See [docs/spec-driven.md](docs/spec-driven.md) for the full methodology)

To install the CLI:
```bash
uv tool install specify-cli --from git+https://github.com/github/spec-kit.git
```
